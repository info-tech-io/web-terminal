  Задание: Система логирования
══════════════════════════════════════════════════

Реализуй два скрипта:

1. log.sh — записывает сообщение в лог-файл
   Использование: bash log.sh <уровень> <сообщение>
   Уровни: INFO, WARN, ERROR
   Формат строки в файле app.log:
     [2024-01-15 10:30:00] [INFO] Запуск приложения

2. tail_log.sh — показывает последние N строк из app.log
   Использование: bash tail_log.sh [N]
   По умолчанию N=10, если аргумент не передан

Примеры:
  $ bash log.sh INFO  "Запуск"
  $ bash log.sh ERROR "Ошибка подключения"
  $ bash tail_log.sh 5

Файлы: ~/exercise/log.sh и ~/exercise/tail_log.sh

Для проверки введи команду: check
