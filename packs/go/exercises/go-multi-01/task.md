  Задание: Пакет mathutils
══════════════════════════════════════════════════

Создай пакет mathutils с тремя функциями для работы
со слайсами целых чисел:

  Max(nums []int) int  — максимальный элемент
  Min(nums []int) int  — минимальный элемент
  Sum(nums []int) int  — сумма элементов

Каждая функция должна возвращать 0 для пустого слайса.

Файл для работы: ~/exercise/mathutils/mathutils.go
Пример использования: ~/exercise/main.go

Запусти тесты вручную: go test ./...
Или для проверки введи: check
