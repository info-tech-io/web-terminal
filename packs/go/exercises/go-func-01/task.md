  Задание: Реверс строки
══════════════════════════════════════════════════

Реализуй функцию Reverse(s string) string,
которая возвращает строку в обратном порядке.
Функция должна корректно работать с ASCII-строками.

Примеры:
  Reverse("hello")  → "olleh"
  Reverse("")       → ""
  Reverse("a")      → "a"
  Reverse("abcd")   → "dcba"

Файл для работы: ~/exercise/solution.go

Запусти тесты вручную: go test
Или для проверки введи: check
