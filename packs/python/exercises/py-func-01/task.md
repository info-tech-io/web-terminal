  Задание: Палиндром
══════════════════════════════════════════════════

Реализуй функцию is_palindrome(s: str) -> bool,
которая возвращает True если строка является
палиндромом (регистр игнорируется, пробелы учитываются).

Примеры:
  is_palindrome("racecar")  → True
  is_palindrome("hello")    → False
  is_palindrome("Madam")    → True
  is_palindrome("")         → True

Файл для работы: ~/exercise/solution.py

Запусти тесты вручную: python3 -m pytest tests/ -v
Или для проверки введи: check
