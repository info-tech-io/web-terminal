  Задание: Калькулятор
══════════════════════════════════════════════════

Реализуй модуль calculator.py с четырьмя функциями:
  add(a, b)       → сложение
  subtract(a, b)  → вычитание
  multiply(a, b)  → умножение
  divide(a, b)    → деление; при b==0 → ValueError

Примеры:
  add(2, 3)       → 5
  subtract(10, 4) → 6
  multiply(3, 4)  → 12
  divide(10, 2)   → 5.0
  divide(1, 0)    → ValueError

Файл для работы: ~/exercise/calculator.py
Пример использования: ~/exercise/main.py

Запусти тесты вручную: python3 -m pytest tests/ -v
Или для проверки введи: check
