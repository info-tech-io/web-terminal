# Child 6: Главная страница — Список упражнений

**Type**: Issue
**Status**: Planned
**Epic**: Epic #2
**Blocked by**: Child 2, Child 3, Child 4 (паки), Child 5 (виджет)

## Problem Statement

Сейчас главная страница (`/`) показывает пустой терминал. Нужна страница со списком
упражнений по всем пакам: клик на упражнение открывает виджет и стартует сессию.

## Solution Overview

Шаблон `index.html` переписывается: JavaScript запрашивает `GET /api/packs`, строит
список упражнений. По клику — создаёт контейнер для виджета, монтирует его с нужными
`data-pack` и `data-exercise`, виджет сам подключается (Child 5).

### Frontend (index.html)
- [ ] Убрать прямое встраивание виджета из шаблона
- [ ] Добавить секцию списка упражнений
- [ ] JS: `fetch('/api/packs')` → построить список
- [ ] По клику: завершить предыдущую сессию (если есть), создать div, смонтировать виджет
- [ ] Передать `data-pack`, `data-exercise`, `data-tps-url` виджету
- [ ] Прокрутить страницу к виджету после монтирования

### Стиль
Минималистичный: без фреймворков, чистый CSS. Акцент на функциональность,
не на внешний вид — это тестовая страница для проверки системы.
